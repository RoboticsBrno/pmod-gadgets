# RFID modul

Modul k připojení RFID čtečky k PMOD konektoru. Použitá čtečka je [RFID-RC522](https://www.laskakit.cz/rfid-ctecka-s-vestavenou-antenou-mfrc-522-rc522/) nebo [MINI RFID-RC522](https://www.vokolo.cz/ctecka-rfid-rc522-mini/).

Jsou potřeba 4 piny + napájení:
- VCC (3.3V)
- GND (zem)
- SCK (Serial Clock)
- MISO (Master In Slave Out)
- MOSI (Master Out Slave In)
- SS (Slave Select)
